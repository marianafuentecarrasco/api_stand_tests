# Pruebas de Soporte de API
Este repositorio contiene los scripts de prueba para la automatización de la API de la aplicación.

### Tecnologías utilizadas
* Python
* Pytest
* Requests

### Cómo ejecutar las pruebas
1. Clonar el repositorio.
2. Instalar dependencias: `pip install requests`.
3. Ejecutar: `pytest`.
